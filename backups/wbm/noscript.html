JavaScript is not enabled

Unfortunately JavaScript is not enabled in your browser.
Please enable JavaScript. You will then be able to utilize the full range of functions of the Phoenix Contact website.

 

Instructions: how to enable JavaScript in your browser:

Firefox

  • In the menu bar, select Tools > Options.
  • In the Options window, select the Content tab.
  • In the Content tab, enable the checkbox for Enable JavaScript.
  • Confirm with OK. JavaScript is now enabled in Firefox.

Internet Explorer

  • In the menu bar, select Tools > Internet Options.
  • In the Internet Options window, select the Security tab.
  • Select the Internet zone. Click on Custom level....
  • In the Security Settings, set the options under Scripting to Enable.
  • Confirm twice with OK.
  • Reload the page with Enable. JavaScript is now enabled in Internet Explorer.
